Servo Motor Biped with Trajectory Optimisation Control

The objectives of this project were to:

  • Design, build and program a servo-motor bipedal robot
  • Develop a kinematic model of the robot
  • Use trajectory optimisation to generate walking gaits based on the kinematic model
  • Develop controllers to implement the generated gaits on the physical robot
Additionally, a support rig was also constructed to assist in testing the functionality of the system.

This project made use of various skills such as 3D design, modelling and simulation. The use of 3D printing, circuit design, soldering, Python programming and microcontroller programming were additional skills that were refined during the course of this project.
Specfic focus was placed on using Pyomo for determing and simulating the optimal walking gaits, that were generated through trajectory optimisation using 3-point collocation.
