Health Monitoring System

This was an electrical design project with the aim of designing and simulating a health monitoring system comprising a temperature sensor and heart rate sensor. The project was unable to be physically realised as it was undertaken during the Covid-19 lockdown period. Resultantly, the system consisted of simulated analog inputs and simulated electrical components, and was evaluated using a simulated testbench.

The project made use of various skills such as analogue circuit design, simulation testing and signal processing.
